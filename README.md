## Rodando a aplicação

Clone esse repositorio

```bash
  git clone https://github.com/LucsGomes/TicTacToe.git
```

Va para a pasta do projeto

```bash
  cd TicTacToe
```

Instale as dependencias

```bash
  npm i | npm install
```

Inicie a Aplicação

```bash
 npm run dev
```
